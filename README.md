# Lista de Tarefas

## ✔️ Descrição
  Um projeto simples pra você organizar suas tarefas diárias sem complicação. você pode adicionar, marcar como concluídas e apagar suas tarefas.
  Feito com o básico da web: HTML, CSS e JavaScript puro.

## 📌 Funcionalidades

  Pra adicionar: Digite sua tarefa e clique em "Criar" ou aperte Enter.

  Pra concluir: É só clicar na caixa de seleção que a tarefa será riscada.

  Pra apagar: Clicou no botão "Apagar", a tarefa já era.
